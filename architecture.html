Layer Technology Purpose

Auth Firebase Auth Email/password, Google, Microsoft OAuth
Deployment Cloud Run (backend), Firebase Hosting (frontends) Production hosting
Dev Tools Docker Compose, Playwright, pytest Local dev & testing

4. Firestore Collections

Collection Purpose Key Fields
users Shared auth profile uid, email, role, displayName
professors Teacher-specific data uid, department, institution
students Student-specific data uid, enrolledClasses, displayName
classes Course definitions classId, professorId, name, code
enrollments Class membership junction studentId, classId, enrolledAt
arcs Story arcs per class arcId, classId, title, objectives, rubric
scenes Individual scenes in an arc sceneId, arcId, order, type, content
character_pools Available NPC characters poolId, arcId, name, personality, role
student_scene_assignments Per-student scene state studentId, sceneId, status, responses
student_character_mappings Student-NPC pairings studentId, characterId, arcId
reasoning_traces AI decision audit trail traceId, sceneId, input, output, model
arc_journals Student reflection entries journalId, studentId, arcId, entries[]
arc_endings Arc conclusion records endingId, studentId, arcId, branch, score
