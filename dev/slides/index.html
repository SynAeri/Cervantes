The Problem

Traditional Assessments Are Broken

  • High-stakes exams measure recall under pressure, not genuine understanding
  • Rigid formats reduce rich learning to multiple choice and short answers
  • No reasoning visibility — educators see scores, never thought processes
  • Student anxiety distorts performance and undermines learning outcomes

The Gap

Universities need assessments that reveal how students think — not just what they remember.

Current tools offer no middle ground between hand-graded essays and automated quizzes.

Our Solution

Assessment Through Interactive Narrative

Cervantes transforms assessment into Visual Novel experiences — curriculum-aligned, AI-generated scenarios where students demonstrate understanding through dialogue and decision-making.

Tech Highlights

What Makes It Work

Curriculum Grounding

CurricuLLM ensures every generated scenario maps to real curriculum standards — no hallucinated assessments

Real-Time Traces

Firestore listeners push live reasoning data to teacher dashboards as students progress through scenarios

Adaptive Dialogue

Gemini-powered Socratic engine adjusts questioning depth based on student responses in real-time

PWA-First Mobile

Students access VN assessments on any device via Progressive Web App — no app store, no installation

3 Core Endpoints

Arc Setup (teacher upload + CurricuLLM), Narrative Generation (Gemini scenes), Dialogue (real-time student interaction)
