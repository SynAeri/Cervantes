API Reference

Complete endpoint reference for the Cervantes backend. All routes are served by the FastAPI application and documented below by feature module.

Base URL https://cervantes-api.azurewebsites.net
Authentication Roles
Professor Firebase token, role=professor
Student Student ID or Firebase token
Authenticated Any valid token
Public No auth required
