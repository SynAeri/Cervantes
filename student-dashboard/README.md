# La Mancha Student Dashboard

Student-facing PWA for the La Mancha narrative formative assessment platform.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack

Same as teacher-dashboard:
- Next.js 16 (App Router with Turbopack)
- TypeScript
- Tailwind CSS v4
- Material Design 3
- PWA support
